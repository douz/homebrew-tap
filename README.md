# homebrew-tap

Homebrew tap for douz open source tools.

## Install

```bash
brew tap douz/tap
```

## Available Formulae

- `mac-motd` (managed by automation from `douz/mac-motd` tags)
- `ismc` (credit: original project [`dkorunic/iSMC`](https://github.com/dkorunic/iSMC); this formula only installs the prebuilt `iSMC` binary published by that upstream project)

## Install Formulae

```bash
brew install douz/tap/mac-motd
brew install douz/tap/ismc
```
